# samuel-ash

A mobile calculator app I built using React Native + Expo.

## What it does

Basic arithmetic — addition, subtraction, multiplication, division — with a clean display that shows your current equation as you type. There's also a fix for the classic 0.1 + 0.2 floating-point issue.

## Stack

- React Native
- Expo Router
- TypeScript

## Running it

Make sure you have Node installed, then:

```
npm install
npx expo start
```

Open it in Expo Go on your phone or press `w` for the browser version.

## Notes

This was built as part of the CMP Git assignment. I customised the colour scheme and layout to make it my own.
